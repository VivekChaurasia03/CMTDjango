**CorrectMyText**
A web based application to perform user selected operations on analyzing and editing the text data in Django backend.

What Can We do from CorrectMyText ?
 - **Remove Punctuations**
 - **UPPERCASE**
 - **New Line Remove**
 - **Extra Spaces Remover**
 - **Numbers Remover**

Requirments
- Python3
- Django
- HTML
